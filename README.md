# CivicHub

## Community Issue Reporting System

CivicHub là hệ thống hỗ trợ tiếp nhận, quản lý và theo dõi phản ánh cộng đồng.

## Tech Stack

### Mobile
- Flutter

### Admin
- React (CoreUI)

### Backend
- Spring Boot

### Database
- PostgreSQL
